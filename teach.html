HTML简介

HTML（超文本标记语言）是用于创建网页的标准标记语言。它不是一种编程语言，而是一种告诉浏览器如何展示内容的标记语言。

你知道吗？ 第一个HTML规范由蒂姆·伯纳斯-李于1991年提出，最初只有18个标签。

HTML如何工作？

HTML由一系列元素组成，这些元素用于封装不同部分的内容，使其以特定方式显示或执行特定功能。

基础HTML元素结构
<tagname attribute="value">内容</tagname>

HTML文档由嵌套的HTML元素构成，整个文档从<html>开始，到</html>结束。

HTML文档结构

每个HTML文档都有固定的基本结构：

<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>页面标题</title> </head> <body> <h1>我的第一个标题</h1> <p>我的第一个段落</p> </body> </html>

重要标签说明：

  • <!DOCTYPE html> - 声明文档类型
  • <html> - 根元素
  • <head> - 包含元数据和页面标题
  • <meta charset="UTF-8"> - 设置字符编码
  • <title> - 定义浏览器标签页标题
  • <body> - 包含所有可见内容

链接与图片

链接和图片是网页中最常用的元素：

<!-- 链接 --> <a href="https://example.com" target="_blank">访问示例网站</a> <!-- 图片 --> <img src="image.jpg" alt="图片描述" width="300">

最佳实践：

  • 始终为图片提供alt属性，这对可访问性和SEO很重要
  • 使用target="_blank"使链接在新标签页打开
  • 使用相对路径链接站内资源

列表

HTML支持有序列表、无序列表和定义列表：

列表类型示例
<ul> <!-- 无序列表 --> <li>列表项 1</li> <li>列表项 2</li> </ul> <ol> <!-- 有序列表 --> <li>第一项</li> <li>第二项</li> </ol> <dl> <!-- 定义列表 --> <dt>HTML</dt> <dd>超文本标记语言</dd> <dt>CSS</dt> <dd>层叠样式表</dd> </dl>

表格

表格用于展示行列数据：

<table> <tr> <!-- 表行 --> <th>姓名</th> <!-- 表头 --> <th>年龄</th> </tr> <tr> <td>张三</td> <!-- 表格数据 --> <td>28</td> </tr> <tr> <td>李四</td> <td>32</td> </tr> </table>

表格效果：

姓名 年龄 职业
张三 28 前端工程师
李四 32 UI设计师

表单

表单用于收集用户输入：

<form action="/submit" method="POST"> <label for="name">姓名:</label> <input type="text" id="name" name="name" required><br> <label for="email">邮箱:</label> <input type="email" id="email" name="email" required><br> <label for="message">留言:</label><br> <textarea id="message" name="message" rows="4"></textarea><br> <button type="submit">提交</button> </form>

语义化标签 (HTML5)

HTML5引入了语义化标签，使网页结构更清晰：

<header>...</header> <nav>...</nav> <main> <article> <section>...</section> </article> <aside>...</aside> </main> <footer>...</footer>

语义化标签的优点：

  • 提升可访问性（屏幕阅读器）
  • 改善SEO（搜索引擎优化）
  • 代码更易读和维护
  • 未来兼容性更好
